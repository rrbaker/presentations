What is Git?

"A distributed revision control and source code management (SCM) system" --Wikipedia

What is Git?

Translation:
A means for nerds to save versions of their code and improve work between teams and also share their progress with others for review, comment, and contributions without losing control of their original work.

A little history & context

The basics

If you've ever used "Track Changes" in Microsoft Word,
then you already understand the basics of git!

Par exemple...

That's git!

In Geek Speak...

  • Nerdy nerds write code and use git to manage each version.
  • Other nerds can clone ("Save as...") their work and can make edits.
  • Those edits can be given back to the original version called pull requests ("track changes edits")
  • Those edits can come back into the code, or not
  • All of this is enabling collaboration without being destructive to the original work

Enter Github

Github.com
